# vue-ssr-instagram

> A Instagram clone project with Vue.js 2.0 using server side rendering.

## Requirements and Use

``` bash
git clone https://github.com/jlopezxs/vue-ssr-instagram.git
cd vue-ssr-instagram

npm install
npm run build
npm start

npm run dev
```
